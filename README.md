# CityPulse AI – Artificial Intelligence CA

This repository contains the implementation for the Artificial Intelligence Continuous Assessment
at the National College of Ireland.

## Project Overview
CityPulse AI is an intelligent agent system that:
- Categorizes news articles using a Naive Bayes classifier
- Finds optimal routes within cities using search algorithms (A*, UCS)
- Optimizes inter-city travel using a Genetic Algorithm (TSP)

## Technologies
- Python 3.11
- Jupyter Notebooks (VS Code)
- scikit-learn
- rdflib

## Structure
- `notebooks/` – Jupyter notebooks for each CA question
- `data/` – Provided datasets (CSV, RDF/XML)
- `models/` – Saved machine learning models

## Notes
This repository is intended for academic submission and learning purposes.
